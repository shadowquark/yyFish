魚五四

2019-04-24

我個人對於蘋果公司的圍牆花園是一直沒有好感的，所以即使用了蘋果的硬件，也儘量使用沒有圍牆的那部分生態。一方面是我對於用戶體驗沒有那麼執着的追求，不會天天琢磨着如何比別人更優雅地使用一個破工具，有個 tmux + vim 就得了。另一方面是我總是需要一些圍牆之外的東西，比如全功能、不帶蘋果牆的 Telegram。這裏面有個有趣的問題，最近在網上看到了一批人反對企業隱藏 url，因爲他們認爲 url 代表了用戶在互聯網面前的知情權和自主權。有這樣看法的用戶在國內不在少數，因爲國內隱藏 url 的主力軍是 BAT 360 這樣的互聯網公司。可問題是，這些用戶中有相當一部分是所有設備都用蘋果的圍牆花園用戶，這就非常諷刺了。比如這些用戶即使需要一個帶手寫筆的觸控設備，在蘋果出之前絕不會去買。在蘋果推出後，如果它們購買了，就會感慨觸控筆真的好用啊，太符合自己的需求了，而這正是大衆對 url 的態度。對於大衆用戶來說，隱藏 url 本來就降低了訪問到不安全、不健康網站的潛在風險，而傻傻地記憶、輸入、複製粘貼 url 也遠不如點擊分享按鈕更符合大衆的使用習慣。換句話說，使用 url 輸入地址前往一個未知網站，就像是買一款陌生品牌的觸控筆產品一樣，可能完全滿足自身的需求，但就是不願意去嘗試，因爲不知道、不信任、不熟悉、不願冒險。所以隱藏 url 對大衆用戶來說，正如增加用戶體驗的圍牆花園。而這時候，一部分已經在圍牆花園中待出自豪感的用戶，去指責隱藏 url 的企業就讓人覺得頗爲搞笑。事實上，隱藏 url 幾乎已經贏得了市場，連 chrome 都在嘗試這麼做了，而那種追求自我管理所帶來的自主權的市場分支，在各個領域都是不斷萎縮的。我並不反感這樣的市場趨勢，只是反感那些在這家圍牆花園裏，嘲笑那家圍牆花園的人，這樣的人很無聊。這些人自以爲自己獨立思考，其實只是會選擇個不同 logo 標的圍牆花園。至於我們要不要守護那一點點自主權，這是個人選擇，我無疑置喙，只是如果這個問題不是在消費品領域，而是在政治領域呢，它就不幸地成了公共問題。我們要不要用自我管理的能力，去換取更多的自主空間呢。這個問題永遠存在，我無法代替別人作選擇，只不過，我自己已經做了選擇。而且我看不起那些反對公權力推進某種價值，卻贊成公權力推進另一些價值的人，這樣的人我最爲厭惡。尤其是這些根據 logo 標選擇了一種價值的人，再自詡獨立思考，便令我在厭惡之上，又多了一份噁心。
