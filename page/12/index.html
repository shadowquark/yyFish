魚五六

最近不知道為什麼突然想起了很多小時候的事，使人非常悲傷。甚至可以說，已經很久沒這麼悲傷過了。小時候自己做過很多不好的事，並不是我覺得它們不好，而是別人這樣覺得。別人覺得不好的事，過了十幾年，那些覺得它們沒什麼的人，也慢慢覺得它們不好起來。那就是最使我悲傷的事情吧，終於他們都長大成人了，只留下了我。

魚五四

我個人對於蘋果公司的圍牆花園是一直沒有好感的，所以即使用了蘋果的硬件，也儘量使用沒有圍牆的那部分生態。一方面是我對於用戶體驗沒有那麼執着的追求，不會天天琢磨着如何比別人更優雅地使用一個破工具，有個 tmux + vim 就得了。另一方面是我總是需要一些圍牆之外的東西，比如全功能、不帶蘋果牆的 Telegram。這裏面有個有趣的問題，最近在網上看到了一批人反對企業隱藏 url，因爲他們認爲 url 代表了用戶在互聯網面前的知情權和自主權。有這樣看法的用戶在國內不在少數，因爲國內隱藏 url 的主力軍是 BAT 360 這樣的互聯網公司。可問題是，這些用戶中有相當一部分是所有設備都用蘋果的圍牆花園用戶，這就非常諷刺了。比如這些用戶即使需要一個帶手寫筆的觸控設備，在蘋果出之前絕不會去買。在蘋果推出後，如果它們購買了，就會感慨觸控筆真的好用啊，太符合自己的需求了，而這正是大衆對 url 的態度。對於大衆用戶來說，隱藏 url 本來就降低了訪問到不安全、不健康網站的潛在風險，而傻傻地記憶、輸入、複製粘貼 url 也遠不如點擊分享按鈕更符合大衆的使用習慣。換句話說，使用 url 輸入地址前往一個未知網站，就像是買一款陌生品牌的觸控筆產品一樣，可能完全滿足自身的需求，但就是不願意去嘗試，因爲不知道、不信任、不熟悉、不願冒險。所以隱藏 url 對大衆用戶來說，正如增加用戶體驗的圍牆花園。而這時候，一部分已經在圍牆花園中待出自豪感的用戶，去指責隱藏 url 的企業就讓人覺得頗爲搞笑。事實上，隱藏 url 幾乎已經贏得了市場，連 chrome 都在嘗試這麼做了，而那種追求自我管理所帶來的自主權的市場分支，在各個領域都是不斷萎縮的。我並不反感這樣的市場趨勢，只是反感那些在這家圍牆花園裏，嘲笑那家圍牆花園的人，這樣的人很無聊。這些人自以爲自己獨立思考，其實只是會選擇個不同 logo 標的圍牆花園。至於我們要不要守護那一點點自主權，這是個人選擇，我無疑置喙，只是如果這個問題不是在消費品領域，而是在政治領域呢，它就不幸地成了公共問題。我們要不要用自我管理的能力，去換取更多的自主空間呢。這個問題永遠存在，我無法代替別人作選擇，只不過，我自己已經做了選擇。而且我看不起那些反對公權力推進某種價值，卻贊成公權力推進另一些價值的人，這樣的人我最爲厭惡。尤其是這些根據 logo 標選擇了一種價值的人，再自詡獨立思考，便令我在厭惡之上，又多了一份噁心。

十幾年過去了，聽到公共道德規範的時候，還是和十三歲的時候一樣厭惡。既厭惡公共道德規範本身，也厭惡推介它們的人。大概在青春期的時候，我從絕對道德中走出來，成了一個只相信自律道德的人，奠定了今天我參與人類之公共活動的最核心基礎。如果這世界上有我必須居於其中的絕對道德大廈，我定默不作聲地抽取其磚磚瓦瓦，直到它轟然倒塌。作爲想生活在別處的人，絕對道德大廈的倒掉，我永遠要奉上一句『活該』。

魚五一

996.ICU 這事特有意思。打開豆瓣，看到大量富有人文關懷的文科從業者，大談歷史的進程、勞動者的尊嚴、社會的病態，各種主義、綜合症滿天飛。打開知乎，各種海外碼農覺得中國全是 996，回去的人腦子都有洞，自己的決策英明神武，週末可以滑雪、吃海鮮，每天可以去健身房，工作生活兩不誤。然後我認真的看了看 996.ICU 這項目，發現人家不但有 996 這個 repo，還有好些 955 的 repo，旨在告訴大家國內哪些企業是 996，哪些是 995。既沒有說國內全是 996，也沒有說選擇 996 的人都沒有尊嚴、患上了斯德哥爾摩綜合症。歸根結底的出發點是幫助大家找到 955 的工作，讓 996 的企業受損以被迫適應市場。無論是豆瓣的人文學者，還是知乎的海外程序員，都絕對算得上社會的高級人才了。在面臨一個嚴肅的社會問題的時候，幾乎脫離不開盲目得可笑的自我優越，別人永遠是需要憐憫和拯救的弱者，自己永遠光鮮值得炫耀，別人的選擇不是病態就是被迫，自己的選擇不是有尊嚴就是更進步。自己永遠有能力代表人類文明的最高峰審視社會上的其他個體，並發表一些指導意見。這就是當代的所謂人文關懷了。祝這種關懷早日腐爛掉，我寧可生活在人文荒漠中。

魚五十

在豆瓣的時候，有幾位師給我留下了很深的印象。他們好像會不少種語言，也好像讀過很多書，而且對各個國家、各個文明的人物和思想，似乎都了如執掌。然而他們從來不能用哪怕稍微複雜一點的思維、思考並表達任何一個具體的問題，只能無限制地重複自己所知道的主義、大師、價值判斷。他們表現自己知識水平的方式總是，文化比較、思想排序、褒貶古人，卻幾乎不能本着任何一種語言獨立地討論一個稍微複雜一點的問題。他們在旁徵博引、好像什麼思想都懂的時候，給我一種感覺，就像是學物理的學生，粒子、原分、凝聚態、天體、宇宙學、實驗、理論、計算，無所不知，但只記住了所有物理學家和他們的成就，自己卻連高中物理題都算不明白，幾乎推導不出任何一個定理。這種豆瓣名師，及他們的崇拜者，真是閃爍着一種獨有的傻氣。

魚四九

魚以爲有些人爲學，貪好博大以至於到了愚蠢的程度、而不自知，實令人惋惜。言及古代政治，只知着眼皇帝及皇帝下一級權力機構，對郡縣之運轉幾一無所知。會幾句中文就動輒議論一切漢文學、漢文化。會一點梵文便慾窮盡印度的一切哲學和宗教。最喜歡以人類文明古往今來的最高視角，對人類的文化進行大視角的概覽，指點江山、激揚文字、褒貶古人。殊不知自己的智識可能連羣論都學不會，對於世間種種甚深微妙法無絲毫深入之理解、甚至尚無能力去理解。唯仗鋪張之詞、博學之態，於網上放二三狂言，得豎子之謬譽而自娛。止增笑爾。魚有幸學了十多年物理，爲了學這十多年物理，又斷斷續續地學了十多年數學，最能深切地感覺到自身智識的邊界。自知世間實有魚智不可及之學，自知人類之諸文明皆有超越今日之魚的不可思量處，斷不敢自居博大而妄論之。魚之生尚有年。願學焉。如是矣。

魚四八

公共場所禁菸 = 禁止公共場所出現感冒的人，理由都是這些人有一定概率使其它人不健康。公共場所用吸菸室控菸 = 要求感冒的人必須在公共場所帶口罩，都是要將對他人引起的潛在危害降到最低。公共場所隨便吸菸 = 公共場所打噴嚏、咳嗽不需要用手唔嘴，都是不在乎他人的看法、以及對他人的潛在危害。但是爲什麼社會選擇的結果卻截然不同呢。因爲幾乎每個人都會感冒，很少感冒、感冒了就呆在家裏休息的人佔少數，而不吸菸的人佔大多數。故而這是一個典型的多數人暴政的例子。事實上兩者還有一個區別，就是感冒的人不需要向其它人繳納任何補償、還可能要花掉一部分醫保，但是吸菸的人卻已經繳納了數額巨大的稅款、還用更短的預期壽命節約了醫保和社保。所以說，多數人暴政是常態，想買通多數人暴政、往往需要繳納高額的費用，還要忍受道德上的羞辱。我從不認爲這種政治形態在任何意義上屬於文明，也不覺得訴求這種政治的人能創造一個美好的未來。

魚四七

科學分爲兩類，一類是物理學，一句是集郵。這句話說起來很美，很多人都知道這句話。但是好多人就因此相信了真的有個美好的物理學。其實物理學的大部分工作也是集郵，唯一的區別就是，集郵的時候懷有一種信仰，即我們在未來一定會發現郵票背後那些優美而簡單的規律的。其實可能幾十年、上百年也發現不了，這麼讓自己相信而已。光譜還是要看的、simulation 還是照做的，磨粉的時候也要假裝我們以後總能找到背後的理論，然而其實就是在集郵罷了。那個抽象出來的美好的物理學，其實是人類對科學的願景，而絕不是事實。
